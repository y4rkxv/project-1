# Сучасна Пекарня (Bakery Landing Page)

Адаптивний лендінг для мережі крафтових пекарень. Проєкт реалізовано з
використанням семантичного HTML, SCSS та JavaScript.

## Технічний стек

- **HTML5:** Семантична розмітка (`<section>`, `<article>`, `<nav>`).
- **CSS / SCSS:** Використання препроцесора SASS для стилізації.
- **Методологія BEM:** Організація класів для підтримки чистоти коду (напр.,
  `.header__container`).
- **JavaScript (ES6):** Реалізація інтерактивних елементів.
- **Swiper.js:** Бібліотека для слайдера у секції "Формати".

## Реалізований функціонал

1. **Адаптивність:** Сайт повністю адаптований під мобільні пристрої, планшети
   та десктоп (Mobile First).
2. **Мобільне меню:** Бургер-меню для навігації на телефонах.
3. **Слайдер:** Карусель форматів пекарень на базі Swiper.js.
4. **Модальне вікно:** Спливаюча форма "Замовити дзвінок" з бекдропом.
5. **Валідація форм:** HTML-валідація полів вводу (телефон, email) та чекбоксу
   політики конфіденційності.
6. **Векторна графіка:** Використання SVG-спрайтів (`icons.svg`) для іконок.

## Структура проєкту

- `index.html` - головна сторінка.
- `scss/` - вихідні файли стилів (розбиті на компоненти та утиліти).
- `css/` - скомпільовані та мініфіковані стилі (`main.min.css`).
- `js/` - скрипти для модалки, меню та слайдера.
- `images/` - оптимізовані зображення та SVG.

## Як запустити

1. Завантажити репозиторій.
2. Відкрити файл `index.html` у браузері.
